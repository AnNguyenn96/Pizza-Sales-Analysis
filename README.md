## 📊 Data Source

<p align="center">
    <img src="Payroll Dataset /source_relational_model.png" alt="source-relational-model" style="border-radius: 10px;">
    </br>
  Source Relational Model
</p>

The dataset contains transactional pizza sales data for the year 2015. Each row represents a single pizza item within an order.

It includes:

- **Order Information**: Order ID, date, time, and order datetime  
- **Product Details**: Pizza name, category, size  
- **Sales Metrics**: Quantity sold and price  
- **Time Attributes**: Time slots for time-based analysis  
- **Customer Behavior**: Order type (single vs multi-item)  
- **Ingredients**: Ingredient list for each pizza  

The dataset is highly denormalized, requiring transformation and normalization before building a proper analytical model.
